# JARVIS – AI Life Decision Assistant

## Vision
A self-evolving AI system that helps people make life decisions using planning, 
behavior tracking, real-time world data, and multi-device support.

## Features
- Life goal planning
- Daily habit tracking
- Real-time decision making
- Voice, text, and camera input
- Self-improving intelligence
- Personalized activity suggestions

## Tech Stack (planned)
- Frontend: React, Next.js
- Backend: Node.js / Java Spring Boot
- AI: Python (LLMs, ML models)
- Database: MongoDB / PostgreSQL
- App: Flutter / React Native

## Roadmap
- Phase 1: Core planning engine
- Phase 2: User behavior tracking
- Phase 3: Multi-device sync
- Phase 4: Voice & camera modules
- Phase 5: Self-evolving AI

## Folder Structure
- /docs – documentation
- /src – source code
- /models – AI models
- /apps – mobile apps
